# 《I！我改变了历史》现代穿越者重构设计

## 1. 问题定义

现有产品的 DeepSeek 调用是真实的，但开局单位是“历史议题”而不是“历史瞬间”。卡牌常用“如果某制度更早出现”这类宏观描述，玩家不知道自己是谁、身在何处、还有多久、正在阻止什么。结果是 AI 虽然在实时推演，玩家仍感受不到参与感。

新版核心命题是：

> 我是一个带着现代经验穿越回去的中国人。我不能自由编造世界，只能在一个真实、著名、迫在眉睫的历史瞬间做出选择，然后承担因果。

## 2. 方案比较

### A. 纯 AI 生成历史卡

- 优点：数量无限，能直接读取画像。
- 缺点：首屏慢，史实可靠性和瞬间具体度不稳定，与当前问题同源。

### B. 50 个策展瞬间 + 本地画像匹配 + AI 推演（采用）

- 优点：卡牌秒开，每个节点的日期、人物、地点和真实结局可以人工保证；AI 集中在其擅长的因果叙事。
- 缺点：需要重写卡牌数据，但这是一次性高价值工作。

### C. 本地卡牌 + AI 重排

- 优点：个性化表达更丰富。
- 缺点：为了排五张卡增加一次网络等待，对游戏乐趣的边际收益低。

## 3. 新玩家旅程

### 3.1 第一屏：建立穿越者档案

产品名必须始终显示为 **《I！我改变了历史》**，不得删掉 `I`。

画像为一个单页“穿越者入境档案”，只收集会影响玩法的字段：

- 称呼：2–12 个字符。
- 现代身份：学生 / 产品与管理 / 工程与科技 / 金融与商业 / 内容与创意 / 公共事务。
- 两项现代优势：谈判、组织、技术、商业、写作、战略、法律、医疗。
- 决策本能：谨慎 / 均衡 / 激进。

全部字段都是结构化选择，除称呼外不允许自由 prompt。画像在本地保存，“再改一次历史”保留画像并回到卡牌；玩家可从卡牌页主动更换档案。

### 3.2 第二屏：为你匹配五个历史瞬间

种子库保留 50 张，但全部改为公元后的具体著名事件：

- 至少 18 张中国史，范围为公元 208 年到 1839 年，避开中国近现代政治敏感节点。
- 其余为中国用户熟悉的世界史瞬间，如 1066 年黑斯廷斯、1453 年君士坦丁堡、1517 年宗教改革、1789 年巴士底狱、1815 年滑铁卢、1914 年萨拉热窝、1962 年古巴导弹危机。
- 不使用公元前卡牌。

每张卡必须包含：

- 精确日期或月份。
- 具体地点。
- 玩家此刻替代或辅佐的真实身份。
- 一个可在几分钟或几小时内执行的决策。
- 三条真实历史锨点。
- 一句“真实历史后来如何”。
- 匹配标签，用于画像本地打分。

推荐不调用 AI，而是用可测试的本地打分：现代身份命中 +3，每个优势命中 +4，决策风格命中 +2，中国史保证至少两张，世界史保证至少两张。同分时根据档案 ID 和卡牌 ID 做稳定打散，不让页面每次刷新随机变化。

### 3.3 第三屏起：五幕 AI 推演

玩家选卡后，DeepSeek 实时生成剧情。前端不预写正文和结局，只约束 JSON 结构。

每幕必须额外输出：

- `role`：玩家在这一幕是谁。
- `immediateObjective`：眼前只有一个明确目标。
- `timePressure`：距离结果还有多久。

叙事要求：

- 用第二人称、现在时，从一个具体动作开始。
- 至少出现一个真实人名、一个真实地名和一个可见物件，不得只谈制度、社会、经济等抽象词。
- 三个选项都是立即可执行的动作，对象、资源和风险明确。
- 至少一个选项使用玩家的现代优势，但不为玩家自动选择。
- 第一幕就是卡牌指定的著名历史瞬间，不再生成一个新的模糊背景。

玩家每幕只能在 A/B/C 中选择。删除自定义开局、局内自由 prompt、D 选项和影响力自选。

## 4. 界面设计

保留现有煤黑、新闻纸、朱砂红、青绿和黄色的视觉方向，不做全新视觉风格。产品可读性的关键改动是信息层级：

- 画像页使用入境档案形态，所有选择在一屏内完成。
- 卡牌第一信号是著名事件名和精确日期，第二信号是“你是谁”，第三信号是“你必须现在决定什么”。
- 事件页顶部固定显示“此刻身份”和“剩余时间”，不让玩家忘记自己在玩什么。
- 删除所有“自己写”按钮。
- 结局头版标题和分享文案统一使用《I！我改变了历史》。

## 5. 数据与状态边界

- `TravelerProfile`：称呼、现代身份、两项优势、决策本能。
- `HistorySeed`：扩展精确日期、历史事件名、穿越角色、立即决策、真实结局和匹配标签。
- `GameScenario`：只能是 `{ profile, seed }`，不再接受自由字符串。
- `GamePhase`：增加 `profiling`；首次进入为 `profiling`，完成档案后为 `selecting`。
- 存储 key 升级为 v2，旧 v1 对局不直接恢复，避免旧自由 prompt 状态污染新流程。

## 6. AI 真实驱动边界

下列内容由 DeepSeek 每局实时生成：

- 每幕现场叙事。
- 三个具体行动和各自即时代价。
- 后续因果回收。
- 五幕后的平行 2026。
- 可信度理由、普通人生活和三条因果链。

下列内容不交给 AI：

- 历史瞬间的日期、地点、真实人物和真实结局。
- 画像匹配的排序。
- 玩家已做的选择。
- 历史偏离度。
- 界面状态、存储、音频和导出。

这样既能明确回答“是否真的由 AI 驱动”，又不把历史准确性和核心流程交给不稳定输出。

## 7. 错误处理

- 画像未完成时不允许进入卡牌。
- 模型返回格式漂移时继续使用一次带原上下文的修复；仍失败则保留该幕重试。
- 断网、超时、鉴权失败保留玩家档案和已做选择。
- 重开一局保留画像；更换画像显式清空当前对局。

## 8. 验收标准

- 首屏出现完整的 `I！我改变了历史`，不再直接出现卡牌。
- 完成画像后立即看到五个与优势相关的著名历史瞬间，且至少两个中国史、两个世界史。
- 50 张卡全部为 AD 年份，每张有具体角色、决策和真实结局。
- 局内不存在自由输入、自定义开局或 D 选项。
- 第一幕叙事与卡牌日期、地点、角色一致。
- 真实 DeepSeek 请求返回的三个选项均为可立即执行的具体动作。
- 五幕、即时回响、偏离度、配乐、结局和 PNG 导出仍可用。
- 所有测试、TypeScript 检查和生产构建通过。
